About This Website

This website was created as part of my portfolio to showcase my skills and projects. Below is a brief overview of how I created this website using HTML, CSS, and version control with GitHub. I used Visual Studio Code (VS Code) as my code editor and pushed the website files to GitHub for hosting.

Steps Taken to Create and Host the Website

  1. Setting Up the Development Environment: I started by setting up Visual Studio Code (VS Code), a powerful code editor. I created a new project folder and added the necessary HTML and CSS files.
  2. Writing HTML and CSS: Using HTML, I created the structure of the site (header, sections, footer) and used Bootstrap for responsive design. I then added custom CSS to make the site visually appealing.
  3. Version Control with Git: I initialized a Git repository within my project folder to track changes. I used the following commands:
    • git init – Initializes a new Git repository
    • git add . – Adds all files to staging
    • git commit -m "Initial commit" – Commits the files with a message
  4. Hosting the Website on GitHub: After creating a GitHub account, I created a new repository on GitHub. To push the website to GitHub:
    • git remote add origin <repository-url> – Adds the GitHub repository as the remote
    • git push -u origin master – Pushes the code to GitHub
  5. Deploying with GitHub Pages: Finally, I enabled GitHub Pages in the repository settings to host my website directly from GitHub. The website can be accessed using the URL provided by GitHub Pages.
