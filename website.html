About This Website

This website was created as part of my portfolio to showcase my skills and projects. Below is a brief overview of how I created this website using HTML, CSS, and version control with GitHub. I used Visual Studio Code (VS Code) as my code editor and pushed the website files to GitHub for hosting.

Steps Taken to Create and Host the Website

  1. Setting Up the Development Environment: I started by setting up VS Code. I made sure the folders and files were neat.
  2. Writing HTML and CSS: Using HTML, I created the structure of the site (header, sections, footer) and used Bootstrap for responsive design. I then added custom CSS to make the site visually appealing. I did use some AI to help with some of the code, as front end is not my strong point.
  3. Version Control with Git: I initialized a Git repository within my project folder to track changes.
  4. Hosting the Website on GitHub: After creating a GitHub account, I created a new repository on GitHub.
  5. Deploying with GitHub Pages: Finally, I enabled GitHub Pages in the repository settings to host my website directly from GitHub. The website can be accessed using the URL provided by GitHub Pages.

Issues I Faced

  • Relearning HTML and CSS: I had only worked with these languages in my first role, and this was at an editing level. So I had to use what I knew with the help of some AI.
  • Windows And Linux: I am using a laptop that runs on windows, but GitHub runs on Linux. This did create some errors. Such as the way each handle case sensitivity.
  • Sizing and spacing: I had to use a lot of trial and error to make sure the site was centre and everything was spaced correctly.
  • Debugging: Small syntax errors in HTML or CSS sometimes caused the site to break

Despite these challenges, I gained valuable experience troubleshooting and learning new skills, which made me more confident in working with front-end technologies.
